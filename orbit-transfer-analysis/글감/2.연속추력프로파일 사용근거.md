## 연속추력 프로파일 사용의 타당성 근거

### 1. 이론적 타당성
- **최적제어 이론의 기반**: 연속추력 문제는 Pontryagin의 최대원리, Hamilton-Jacobi-Bellman 방정식 등 확립된 최적제어 이론의 직접적인 적용 대상임
- **해의 존재성 및 유일성**: 적절한 제약조건 하에서 연속추력 최적제어 문제는 수학적으로 well-posed되며, 해의 존재성과 유일성이 보장됨

### 2. 실용적 타당성
- **전기추진 시스템의 발전**: 현대 우주선에서 ion thruster, Hall-effect thruster 등 연속적 저추력 추진 시스템이 실제로 광범위하게 사용됨
- **실제 임무 적용 사례**: Dawn, Hayabusa, BepiColombo 등 다수의 심우주 탐사선이 연속추력을 사용하여 성공적으로 임무 수행

### 3. 모델링의 정확성
- **저추력 시스템의 본질적 특성**: 전기추진 시스템은 높은 비추력(2,000-10,000초)과 낮은 추력(수십 mN~수 N)을 가지므로, 연속적으로 작동하는 것이 물리적으로 합리적
- **이산화 오차의 최소화**: 실제 연속적인 물리 현상을 연속 모델로 표현하면, 이산화로 인한 근사 오차를 원천적으로 배제할 수 있음

### 4. 최적화 관점의 장점
- **해 공간의 풍부함**: 연속 제어 입력은 더 넓은 해 공간을 제공하여 더 나은 최적해를 탐색할 수 있음
- **부드러운 제어 프로파일**: 연속 모델은 자연스럽게 부드러운 제어 입력을 생성하며, 이는 실제 구현 시 제어 시스템의 부담을 줄임

### 5. 수치적 장점
- **현대적 최적화 기법의 적용**: Direct collocation, Gauss pseudospectral method 등 효율적인 수치 최적화 기법이 연속 문제에 직접 적용 가능
- **민감도 분석의 용이성**: 연속 모델에서는 변분법을 통한 민감도 분석과 costate 해석이 자연스럽게 수행됨
