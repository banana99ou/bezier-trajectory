# 딥 리서치 프롬프트: 연속 추력 궤적 최적화 연구 동향

## 연구 목적
LEO-to-LEO 연속 추력 궤도전이에서 최적 추력 프로파일의 유형 분류 및 영향 요인 분석에 관한 논문의 Introduction 작성을 위해 관련 분야 최신 연구 동향을 조사합니다.

---

## 프롬프트 1: 저추력 궤적 최적화 방법론 동향

```
저추력(low-thrust) 우주비행체 궤적 최적화 분야의 최신 연구 동향(2018-2024)을 조사해 주세요.

다음 항목에 대해 구체적인 논문/저자와 함께 정리해 주세요:

1. Direct Collocation 기반 방법론의 최신 발전
   - Hermite-Simpson, Gauss-Lobatto 등 collocation scheme 비교 연구
   - 대규모 문제에 대한 계산 효율성 개선 연구

2. Pseudospectral Methods의 최신 동향
   - Gauss pseudospectral method (GPM)
   - Radau pseudospectral method
   - 최근 개선 사항 및 응용 사례

3. Convex Optimization 기반 접근법
   - Sequential Convex Programming (SCP)
   - Lossless convexification
   - 실시간 최적화 적용 연구

4. 기타 최신 방법론
   - Machine learning / Neural network 기반 접근
   - Differential dynamic programming
   - Hybrid methods

각 방법론의 장단점과 적용 가능 범위도 함께 설명해 주세요.
주요 저널: Journal of Guidance, Control, and Dynamics (JGCD), Acta Astronautica, Aerospace Science and Technology, Journal of Spacecraft and Rockets
```

## 프롬프트 2: LEO 저추력 전이 연구 동향

```
저궤도(LEO) 위성의 저추력 궤도전이에 관한 최신 연구(2018-2024)를 조사해 주세요.

다음 주제에 대해 관련 논문과 함께 정리해 주세요:

1. LEO 위성 군집(constellation) 재배치 및 유지
   - Starlink, OneWeb 등 대규모 군집 운용
   - 다중 위성 동시 기동 최적화

2. 궤도 경사각 변경(plane change) 최적화
   - 저추력을 이용한 면외 기동 연구
   - 면내 기동과 면외 기동의 결합 최적화

3. 우주 잔해(debris) 제거 관련 궤적 설계
   - Active Debris Removal (ADR) 임무
   - 다중 타겟 랑데부 궤적 최적화

4. LEO-to-LEO 전이에서 추력 프로파일 특성에 대한 연구
   - 추력 프로파일의 형태(peak, coasting 등)에 대한 분석
   - 궤도 조건에 따른 최적 기동 전략 비교

특히 "추력 프로파일의 유형"이나 "최적 기동 패턴 분류"에 관한 연구가 있다면 상세히 알려주세요.
```

## 프롬프트 3: 추력 프로파일 특성 및 분류 연구

```
연속 추력(continuous thrust) 궤적 최적화에서 추력 프로파일의 형태 및 특성에 관한 연구를 조사해 주세요.

다음 질문에 답할 수 있는 문헌을 찾아주세요:

1. 최적 추력 프로파일의 유형 분류
   - 단봉형(unimodal, single-peak) vs 다봉형(multimodal, multi-peak) 추력 프로파일
   - Bang-bang control vs smooth control 비교
   - Coasting arc(추력 0 구간)의 발생 조건

2. 추력 프로파일 형태에 영향을 미치는 요인
   - 궤도 요소 차이(고도, 경사각, 이심률 등)의 영향
   - 전이 시간 제약의 영향
   - 비용함수 선택(연료 최소화 vs 에너지 최소화)의 영향

3. 이산 추력(impulsive)과 연속 추력의 비교
   - 호만전이, bi-elliptic 전이의 연속 추력 대응물
   - 임펄스 수 증가에 따른 연속해 수렴 연구
   - 에너지 효율성 비교

4. Parametric study 또는 데이터베이스 기반 연구
   - 다양한 조건에서의 체계적 비교 연구
   - 최적 기동 유형의 regime map 또는 classification diagram

이 분야에서 "어떤 조건에서 어떤 유형의 추력 프로파일이 최적인가"를 분석한 연구가 있는지 특히 중점적으로 조사해 주세요.
```

## 프롬프트 4: 전기추진 시스템 및 실제 임무 동향

```
전기추진(Electric Propulsion) 시스템을 사용한 우주 임무의 최신 동향(2018-2024)을 조사해 주세요.

1. 전기추진 기술 발전
   - Ion thruster, Hall-effect thruster의 최신 성능 지표
   - 추력 수준, 비추력(Isp), 전력 효율 등 기술 발전 현황
   - 차세대 추진 시스템 (VASIMR, Pulsed Plasma 등)

2. 실제 임무 적용 사례 (2018년 이후)
   - 심우주 탐사: BepiColombo, Psyche 등
   - 지구 궤도: Starlink, OneWeb 위성군의 전기추진 사용
   - 정지궤도 위성의 전기추진 활용

3. LEO 위성에서의 전기추진 활용
   - 소형위성/큐브샛용 전기추진 시스템
   - 궤도 유지 및 수명 연장 활용
   - 군집 위성 재배치 활용

4. 연속 추력 궤적 설계의 실무 적용
   - 실제 임무 설계에서 사용되는 최적화 도구 (GMAT, STK, 자체 개발 등)
   - 운용 중 궤적 수정 및 재계획

각 임무에서 사용된 추력 프로파일의 특성(peak 수, coasting 여부 등)에 대한 정보가 있다면 포함해 주세요.
```

## 프롬프트 5: Research Gap 확인

```
다음 연구 주제에 대해 기존 문헌에서 다루어지지 않은 gap을 확인해 주세요:

연구 주제: "LEO-to-LEO 연속 추력 궤도전이에서 다양한 궤도 configuration에 따른 최적 추력 프로파일의 유형 분류 및 지배적 영향 요인 분석"

확인이 필요한 사항:
1. LEO-to-LEO 전이에 대한 체계적인 parametric study가 존재하는가?
2. 추력 프로파일을 peak 수 기반으로 분류한 연구가 있는가?
3. 궤도 파라미터와 추력 프로파일 유형 간의 정량적 관계를 규명한 연구가 있는가?
4. 최적화 수행 전 예상 프로파일 형태를 예측하는 방법론이 제시된 연구가 있는가?

위 질문들에 대한 답변과 함께, 이 연구 주제의 novelty를 뒷받침할 수 있는 문헌적 근거를 제시해 주세요.
```
